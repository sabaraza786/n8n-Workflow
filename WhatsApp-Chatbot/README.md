# 💬 WhatsApp Chatbot using n8n

An AI-powered WhatsApp chatbot built with n8n that automates conversations and responds to user messages.

- 💬 WhatsApp message handling

## 📥 Import Workflow

1. Download `workflow.json`.
2. Open **n8n**.
3. Click **Import from File**.
4. Select `workflow.json`.
5. Configure your API credentials and run the workflow.
