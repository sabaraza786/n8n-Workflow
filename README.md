# 🦷 BrightSmile Dental AI Receptionist

An AI-powered virtual receptionist built with **n8n** that automates patient communication for a dental clinic.

---

## 🚀 Features

- 🤖 AI-powered patient support
- 📅 Appointment scheduling
- 💬 Automated responses to patient queries
- 📧 Email notifications
- 🔄 Workflow automation with n8n

---

## 🛠️ Tech Stack

- n8n
- OpenAI
- WhatsApp API
- Google Calendar
- Gmail
- Webhooks

## 📥 Import Workflow

1. Download `workflow.json`
2. Open **n8n**
3. Click **Import from File**
4. Select `workflow.json`
5. Configure your credentials
6. Activate the workflow
